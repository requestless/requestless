## Installation


### Setup

- Clone the repo
- Install serverless `npm i -g serverless`
- Install dependencies `npm i`
- Add Google Cloud credentials on `credentials.json`
- Create secrets.yml `cp example.secrets.yml secrets.yml`
- Deploy functions `sls deploy`
